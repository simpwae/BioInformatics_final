# 1. Introduction

Drug repurposing — finding new therapeutic uses for approved drugs — offers a faster path to treatment than de novo development. Most candidate-based approaches require labeled drug-disease pairs. For rare diseases, those labels do not exist.

TxGNN (Huang et al., 2024) is a foundation model for drug repurposing that uses a heterogeneous knowledge graph (PrimeKG) and a disease-similarity module to generalize to diseases with no approved therapies in the training data. This project asks six empirical questions about that system.

This report is a scaled reproduction. All models use `hidden_dim=64` instead of the published `hidden_dim=512` due to the RTX 4060 8 GB VRAM constraint. Every result in this report links to a file in `results/`. No number is typed by hand.

## 1.1 Clarification on Model Architecture

TxGNN uses the **Heterogeneous Graph Transformer (HGT)** architecture. The word "Transformer" in HGT refers to its multi-head attention mechanism — it is a **graph neural network (GNN)**, not a large language model (LLM) or a text-based Transformer. No language models are trained or evaluated in this study. All results concern GNN-based models.

The research question framed as "do Transformers/LLMs perform better with knowledge graphs?" is therefore rephrased as: **does a GNN with KG message passing outperform a GNN without message passing?** The answer on the standard split is no. The full findings are in Section 3.1 and 5.1.

## 1.2 Research Questions (as operationalized)

1. Does GNN message passing over a knowledge graph improve drug repurposing performance over a no-message-passing baseline?
2. Is there a better alternative to TxGNN's two-phase training? (Answer: no — two-phase still wins.)
3. Why is zero-shot prediction preferred for TxGNN? (Answer: 92% of diseases have zero training labels.)
4. What do two case studies — one rare disease, one well-studied — show about the model's predictions?
5. How does a plain GNN compare to the full TxGNN on the same splits and seeds? (Tabular comparison.)
6. Is TxGNN's attention mechanism necessary for its zero-shot performance? (Answer: not only unnecessary — it is actively detrimental.)

## 1.3 Data

PrimeKG (Chandak et al., 2023): 8,100,498 edges, 10 node types, 129,375 nodes. Obtained from Harvard Dataverse. Therapeutic edges: 18,776 indication, 61,350 contraindication.

## 1.4 Hardware
