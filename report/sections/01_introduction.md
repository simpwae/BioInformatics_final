# 1. Introduction

Drug repurposing — finding new therapeutic uses for approved drugs — offers a faster path to treatment than de novo development. Most candidate-based approaches require labeled drug-disease pairs. For rare diseases, those labels do not exist.

TxGNN (Huang et al., 2024) is a foundation model for drug repurposing that uses a heterogeneous knowledge graph (PrimeKG) and a disease-similarity module to generalize to diseases with no approved therapies in the training data. This project asks six empirical questions about that system.

This report is a scaled reproduction. All models use `hidden_dim=64` instead of the published `hidden_dim=512` due to the RTX 4060 8 GB VRAM constraint. Every result in this report links to a file in `results/`. No number is typed by hand.

## 1.1 Research Questions

1. Does knowledge graph augmentation improve drug repurposing performance over a no-KG baseline?
2. Is there a better alternative to TxGNN's two-phase training?
3. Why is zero-shot prediction preferred for TxGNN?
4. What do two case studies — one rare disease, one well-studied — show about the model's predictions?
5. How does a plain GNN compare to the full TxGNN on the same splits and seeds?
6. Is TxGNN's attention mechanism necessary for its zero-shot performance?

## 1.2 Data

PrimeKG (Chandak et al., 2023): 8,100,498 edges, 10 node types, 129,375 nodes. Obtained from Harvard Dataverse. Therapeutic edges: 18,776 indication, 61,350 contraindication.

## 1.3 Hardware

NVIDIA GeForce RTX 4060, 8 GB VRAM. All models trained with CUDA. Results labeled `scaled_reproduction` throughout.
